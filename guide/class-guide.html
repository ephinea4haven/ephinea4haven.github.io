Hunter · Ranger · Force

本文分为两层：先讲从 Normal 一路成长到 Ultimate 都适用的 Vanilla PSOBB 基础玩法，再整理 Ephinea 多人终局的装备与分工。 它不是固定配装答案；HU 抢关键目标，RA 控场并覆盖整波怪，FO 放大整队效率，但三者都必须根据手边装备、任务与队伍调整。

展开：Vanilla 定义、四档难度与三职业通用基本功

这里的 Vanilla 指什么

本文中的 Vanilla 指 PSO Blue Burst 原本的战斗框架：三段攻击、命中检定、武器属性与特殊攻击、Mag、材料、技能和四档难度。 它不等同于 Ephinea 的 Classic 模式：Classic 还会关闭 Episode 4 及相关物品，并采用官方 BB 的难度门槛。 下文所有“Vanilla 路线”都不依赖昂贵终局装备，也适合普通 Ephinea 角色开荒。

四档难度的学习重点

阶段主要目标不要急着做的事
Normal试遍武器动作、养 Mag、学会 N/H/S 与敌人硬直为短期装备投入大量 Grinder 或材料
Hard开始保留 Hit 武器、补齐基础技能和范围武器只看红盒星数，忽略绿色普通武器
Very Hard形成“单体、范围、远程、控场”四类工具，准备 Ultimate只堆伤害而不补 ATA、HP 和抗性
Ultimate适应更快的敌人、更高 EVP / 抗性，按区域换武器与战术继续用低 Hit 特殊攻击硬赌命中

Vanilla 开荒的资源优先级

  1. 先养 Mag：HU / RA 先做 POW + DEX，简单成品可用 5/145/50/0 或 5/150/45/0；纯施法 FO 先做 MIND。DEF 通常不值得主动堆。
  2. Hit 比颜色重要：武器的 1% Hit 实际增加 1 ATA，适用于所有敌人。高 Hit 普通武器往往比无 Hit 红盒更实用。
  3. 保留敌种属性：Native、A.Beast、Machine、Dark 会提高对相应敌人的物理伤害；进入新区域前换对属性，收益常高于一点面板 ATP。
  4. 材料先按方案吃：人类共用材料上限 250，纽曼与机器人为 150；HP / TP Material 另算。没决定最终方案前，至少不要随意吃 DEF 和 EVP Material。
  5. 普通武器就是主力：Sword / Partisan、Shot、Slicer、Mechgun 等范围或多段武器决定清图效率；稀有武器主要是补充特殊功能。

先看结论：队伍里该做什么

HU：解决最危险的怪

提前站到刷怪点侧后方，用最高单体伤害优先击杀高血量、会冲锋或会打乱队形的敌人；不要只追着杂兵跑。

站位要求最高近身风险高

RA：让整波怪失去威胁

在能覆盖多数敌人的安全距离开火，以冻结、麻痹、Hell、Demon's 和范围火力切换处理不同波次。

工具箱最全容错较高

FO：提高全队总输出

维持 S/D，快速补上 J/Z、Resta、Anti；能提前叠 Gifoie 就抢节奏，武器或技能更合适时就及时切换。

观察全场切装频繁

三职业通用基本功

  1. 刷怪知识优先于装备：先记危险怪和大波次，再补齐普通波次；知道下一波出现在哪里，才能预站位、预攻击或预叠技能。
  2. 不要重复处理同一目标：看懂队友走向并主动接管无人处理的区域。集中火力并非永远最快，合理分怪通常更有效。
  3. 熟练快捷菜单：战斗中用快捷菜单切武器、吃药和施法；它比翻主菜单安全，也能让动作在可执行的第一帧排队。
  4. 理解连段：N / H / S 分别是普通、重击、特殊。根据怪物硬直和武器收招决定二段或三段，不要机械地打满。
  5. 命中优先：无法稳定命中的理论伤害没有意义。高 Hit 武器、ATA、站位和 SN glitch 都是在解决同一个问题。

现行机制：Ephinea 已移除 Damage Cancel（DMC）。旧 Force 论坛帖中“不要攻击，以免覆盖队友伤害”的相关论述仅有历史参考价值； 现在仍应避免无意义攻击，但原因是站位、控场与分工，而不是 DMC。

HU · Hunter / 战士

HU 的优势是最高级别的近身爆发，短板是有效范围有限。高效 HU 不是“见怪就砍”，而是提前判断哪只怪必须最快消失，再用正确的角度和连段完成 Combo Kill（一个连段内击杀）。

Vanilla 成长路线

时期武器与打法培养重点
Normal–Hard Saber / Dagger 打单体，Sword / Partisan 扫聚团怪，Handgun 处理飞行、陷阱和不宜近身的目标。先用 N-N-H 或 N-H-H 保证命中。 Mag 以 POW 为主、DEX 为辅；优先装备 Arm 类单位。学会绕背、二段收手和延迟第三击。
Very Hard 加入 Mechgun 处理单体，Slicer 处理分散敌人；开始保留高 Hit 的 Charge / Spirit / Berserk 普通武器。 准备一把可靠远程武器和一把范围武器，不要所有场景都用 Sword。
Ultimate 敌人更快且 EVP 更高，先冻、麻或诱导挥空再近身。攻击不中时宁可换高 Hit 低阶武器，也不要硬用无 Hit 稀有武器。 按敌种准备属性武器，逐步加入 Arrest、Hell、Demon's 与高伤牺牲特殊。

Vanilla 关键习惯：不要把高 HP 当成站桩资本。HU 的防御来自距离、角度、硬直与击倒控制；面对包围时先拉成一条线或扇面，再挥范围武器。

四个角色怎么选

HUmar

人类均衡型，近战能力稳定并可使用最高 Lv15 的攻击与恢复技能；不能使用 Shifta / Deband，组队更依赖 FO 支援。

  • 打法：先以 Jellen 降低近战风险、Zalure 提升击杀线，再从侧后方用 Sword / Vjaya / Mechgun 爆发。
  • 单刷：依靠 S-Red's Blade 或 Mylla & Youlla 获得 S/D；没有陷阱，Arrest Slicer 与 Gifoie 是主要控场补丁。
  • 误区：“均衡”不代表样样齐全；没有 S/D 与强控场，HUmar 实际更考验站位和装备。

HUnewearl

HU 中最低 HP、较低 ATP，但拥有 Lv20 S/D/J/Z、Resta 与天然 TP 回复，单刷和自给自足能力最强。

  • 打法：开波前维持 S/D，接近强敌先 J/Z；用 Rabarta / Gifoie 创造近身窗口，再用 Spirit、Berserk 或多段武器收尾。
  • 定位：无 FO 队伍主动补支援；有 FO 时把 TP 留给控场、Resta 与远程补刀，专注 HU 的关键目标。
  • 误区：不要因为会技能就当低配 FO；她的主输出仍是高 Hit 物理武器。

HUcast

全职业最高 ATP，拥有冻结、混乱、伤害陷阱，是最直接的近身爆发选择；没有技能，命中与队伍支援尤其重要。

  • 打法：刷怪点放 Freeze Trap 并立即打爆，趁冻结用 Sword / Vjaya 清群或 Dark Flow / Mechgun 处理高血目标。
  • 陷阱：Freeze 留给危险精英和爆发窗口；普通密集怪优先 Confuse，让它们聚团互打；Damage 可秒 Slime 分裂体。
  • 误区：高 ATP 不等于每次都打满三段；机器人回复品有限，二段安全收手通常更快。

HUcaseal

机器人 HU 中 ATA、EVP 更高，动作性能和陷阱带来稳定控制；ATP 低于 HUcast，更偏灵活与精准。

  • 打法：利用 HU 最高 ATA 更早稳定使用 Arrest / Hell；冻结后以快速 Dagger 动作或多段武器处理高 EVP 单体。
  • 定位：陷阱控场优先于和 HUcast 比面板伤害；主动处理那些 HUcast 特殊攻击容易 Miss 的目标。
  • 误区：高 EVP 不是可靠减伤，仍要用陷阱、绕背与硬直保持主动。

角色专项教程

HUmar · 无陷阱的均衡近战如何控场

核心循环：进房维持安全距离 → 对近战威胁用 Jellen、对高血目标用 Zalure → Gifoie 拦第一轮冲锋 → 从侧后方用 Sword / Slicer 整形 → Mechgun、Vjaya 或 Excalibur 收高血目标。HUmar 的技术不是“什么都会一点”，而是在没有陷阱时用技能换出近战窗口。

单刷

S-Red's Blade 或 Mag PB 补 S/D；先 Jellen 再近身，Resta 留在脱离攻击动画后使用。对一群散怪先 Gifoie 或 Arrest Slicer，不要逐只追砍。

组队

FO 已经上好 J/Z 时不要重复施法，直接接危险目标。无 FO 时主动 Zalure 厚怪，但不必为普通杂兵逐只施法而丢失节奏。

  • 武器优先：高 Hit Raygun、DoB、范围剑、Mechgun，再补 Demon's Mechgun 与 Zalure Gun。
  • 练习重点：Gifoie 出手后立刻切近战武器；练二段安全收手，弥补没有 Freeze Trap 的硬控空缺。
  • 常见失误：边被包围边 Resta；先退出攻击范围再治疗，通常比硬吃下一击更快。
HUnewearl · 支援、自保与物理输出的切换

核心循环：转场补 S/D → 开波对关键怪 J/Z → Rabarta / Gifoie 制造窗口 → 用高 Hit 物理武器完成击杀。Lv20 支援让她单刷完整，但较低 ATP 意味着必须利用属性、特殊攻击和多段命中，而不是拿普通剑硬磨。

  • TP 分配：普通怪只需一次 J/Z 或不施法；TP 优先留给 S/D、强敌 J/Z、Resta 与救场 Anti。天然 TP 回复适合安全转场停留，不适合战斗中原地等。
  • 伤害选择：弱怪用 DoB / Sword；厚怪先 Demon's，再以 Mechgun、Excalibur、Vivienne 收尾；敌人技术抗性低时用技能省去危险贴身。
  • 组队：有 FO 就退回 HU 职责，专打关键目标；无 FO 才承担全套支援。不要为了刷新还剩很久的 S/D 放弃刷怪点站位。
  • 生存：低 HP 要用 Jellen、侧后方和二段连击预防伤害，不能把 Resta 当作事后无限容错。
HUcaseal · 高 ATA 陷阱猎人的控制打法

核心循环：先用 ATA 优势施加 Arrest / Hell → 对剩余威胁 Freeze → 利用快速 Dagger / Twin Sword 动作贴侧面收尾。她不是“小号 HUcast”，优势在更早稳定命中特殊攻击，并用精准控制减少队伍停顿。

  • 分散怪：Arrest J-Cutter / Slicer 先扫一轮，麻痹成功后再决定近战；比直接冲进散怪中心安全。
  • 高 EVP 单体：Freeze 降低 EVP 后用多段武器；第三段杀不掉就二段离开。
  • 陷阱预算：Freeze 增长少于 HUcast，不能照搬每个危险波都冻结的路线；可冻结的普通波交给 Arrest，真正会破阵的目标才花 Freeze。
  • 常见失误：依赖高 EVP 站桩。闪避是概率，不是控制；一旦被多个敌人从不同角度夹击，仍应先撤出包围。

角色深度教程 · HUcast 怎么打

HUcast 的强不是“面板高所以冲进去砍”，而是用刷怪知识把有限射程变成优势：提前占住危险目标的侧后方，用陷阱制造四秒窗口，再在窗口内完成 Combo Kill。 初学时只练安全击杀；熟悉任务后，再练预埋陷阱、精确站位和低血 Dark Flow。

进房前：六个功能位，而不是六把最贵武器

必须常驻

  • Raygun：射爆头顶陷阱、远程补刀、拉单只怪。
  • Sword / Partisan：怪已聚成正面扇区时清群。
  • Slicer：怪分散、会移动，或无法安全贴身时使用。
  • Mechgun / Twin Sword：冻结后的高血单体。

按任务替换

  • Arrest：Dorphon、Girtablulu 等不能冻结或更适合麻痹的目标。
  • Hell：Episode 2 中低暗抗、用物理伤害反而慢的敌人。
  • Twin Blaze：用残留 Gifoie 预设即时爆陷阱，也可挡冲锋。
  • Zalure ES：单刷 Boss 或队伍没有可靠 Zalure 时补位。

快捷栏建议：Freeze Trap、Confuse Trap、Mate、Star Atomizer 放最顺手的位置；武器按“远程 → 范围 → 单体 → 特殊”固定排序。肌肉记忆比背包里多带一把相似武器更有价值。

每一波怪的判断顺序

  1. 先看危险目标：会冲锋、跳跃、远程控制或血量最高的怪优先。队友已经朝它移动，就接管另一侧，不要四个人追同一只。
  2. 再看形状：聚成线或扇面用 Sword；分散在宽角度用 Slicer；一至三只高血目标用 Mechgun、Twin Sword 或 Excalibur。
  3. 决定是否控制：目标能冻结且会破坏队形，放 Freeze；需要先让普通怪聚团可短暂等位；不能冻则尝试 Arrest。不要为了“省陷阱”让整队被冲散。
  4. 从侧后方开打：正面贴脸只会把最高 ATP 换成最高吃伤。先诱导一次挥空，再横移到侧后方，是无陷阱时最稳定的开局。
  5. 击杀线决定二段或三段：第三段杀不掉且收招会被打，就停在第二段、走位重置。第三段不是奖励键。

三种陷阱怎么用

陷阱正确时机引爆方式常见错误
Freeze 危险波、跳怪落点、高 EVP 目标、需要让全队安全爆发时。冻结约四秒，并降低敌人 30% EVP。 临场用 Handgun 射爆；已知刷怪点可预埋；有 Twin Blaze 时先铺 Gifoie，再放陷阱即可自动爆。 怪还没聚拢就立刻冻，导致 Sword / Vjaya 覆盖不到；或对不可冻结目标浪费。
Confuse 普通怪伤害很高、可让它们互打，或需要它们围到一起时；开荒缺伤害时价值更高。 通常让陷阱自然触发，利用混乱期间调整站位或换范围武器。 覆盖队友刚施加的 Freeze；Ultimate 乱放会让移动不可预测，反而破坏预定站位。
Damage Ultimate 秒杀刚生成的 Slime、让 Rappy 倒地逃跑，或给低 EFR 小怪补伤害。 刷怪点预埋最稳定；普通清怪不必把它当主要输出。 把 Damage 当成弱版 Freeze 连续丢，既没控制也浪费动作时间。

连段不是固定答案：先练这五组

输入使用场景判断标准
N-H不熟的近战目标、Sword 正面控群两段后立即横移；不知道第三段是否安全，就默认不打。
N-N-H开荒、Hit 不够、需要稳定命中刻意延迟第二与第三击，让每次硬直真正接续；连按会浪费硬直时间。
N-H-H冻结或已麻痹目标，普通伤害足够第三击必须能击杀，或目标无法在收招时反击。
S-N-SSlicer 特殊攻击，如 DoB、Arrest、Hell中距离利用 Accuracy Glitch 提高特殊攻击命中；距离和投刃返回时机会影响结果。
S-S-S / S-H-H冻结窗口内的 Vjaya、Berserk 或其他高风险特殊只有命中率、HP/金钱成本和击杀线都明确时使用；否则换安全连段。

延迟的含义：敌人正在后仰时再次命中，不会把后仰时间重新加满。正确做法是在连段允许的输入窗口内稍等，让上一段硬直接近结束再出下一段，而不是看到三段按钮就最快连按。

典型局面：看到什么，就切什么

一群普通怪向你走来

后退半步让它们叠成扇面 → Freeze → 从侧面用 Sword。若仍很散，改用 Slicer，不要追着第一只砍散整群。

一只高血怪带数只杂兵

队友能清杂兵时直取高血怪；没人接杂兵则先 Freeze 全波，再用 Excalibur / Vjaya 同时覆盖，或二段拉开重置。

Dorphon / Delbiter 冲锋

不要正面赌连段。预埋陷阱或 Twin Blaze 铺 Gifoie；不可冻的目标改 Arrest，控制成功后贴侧面爆发。

飞行或垂直高度目标

先用 Raygun 保持输出与诱导；熟练后用具有无限垂直高度的 TJS / Dark Flow 特殊处理对应刷怪点，避免在落点下盲等。

单刷与组队的职责差异

场景你要主动承担资源管理
单刷用 S-Red's Blade 补 Lv3 S/D、Zalure ES 处理 Boss/厚甲目标；多用门、墙角和 Confuse 聚怪。记住补给点；每种 Mate 只有 10 个，先用站位省血，不要靠药硬换。
普通组队FO 上完 J/Z 后立即进入击杀位；Freeze 给整队创造共同窗口，优先处理队友慢打的高血目标。Star Atomizer 用来救近身队友和保持节奏；看到队友已控场，不要用 Confuse 覆盖。
熟练速刷按地板标记预站位、预埋陷阱，争取敌人出现即进入击杀帧；通过队友走向分配目标。按房间预算陷阱，不是全程省，也不是每波必放；把 Freeze 留给能明显缩短波次的刷怪点。

低血 Dark Flow：最后学，不是入门门槛

Dark Flow 特殊需要 HP 低于最大值的 12.5%。贴住目标可让五道剑波重叠，形成极高单体爆发；但低血时任何站位错误都可能死亡。 先学会用 Monomate +200 HP、Dimate +500 HP 调整血线，并记住固定伤害，再进入公开队伍练习。队友需要治疗时不要要求全队迁就你的低血玩法；切回普通高血连段仍然是合格 HUcast。

20 分钟练习法

  1. 5 分钟：只拿 Sword，对同一种近战怪练“诱导挥空 → 侧移 → 延迟 N-N-H”，直到不吃反击。
  2. 5 分钟：同一房间分别用自然触发、Raygun 射爆、Twin Blaze 引爆 Freeze，比较自己剩余的有效攻击时间。
  3. 5 分钟：把每波先分成“聚团 / 分散 / 高血单体”，强制切对应武器，不允许一把 Sword 打到底。
  4. 5 分钟：复跑任务，只记三个最危险刷怪点的站位和陷阱时机；下一轮再补三个，不必一次背完整张图。

实战循环

  1. 进房间前确认下一波刷怪点和优先目标。
  2. 站在敌人侧面或背面，避免正面贴身；必要时以地板纹理作为固定站位标记。
  3. 分散敌人用 Slicer，聚团敌人用 Sword / Partisan，高血量单体用 Mechgun、Twin Sword 或 Dark Flow。
  4. 延迟第二、第三击维持硬直；收招不安全或第三击不能击杀时，只打两段。
  5. 用 Gifoie、Twin Blaze、冻结或 Arrest 阻止冲锋，再进入爆发位置。

HP 与低血玩法

HU 经常贴怪，也最常利用低血量 Mag 无敌触发和 Dark Flow 特殊攻击。先掌握固定伤害、药品回复量与安全站位，再练低血线； Monomate 回复 200 HP、Dimate 回复 500 HP，可用来精确调整 Ultimate 下的血量。失败后复盘站位和连段，比盲目堆防御更有效。

武器路线

先准备的通用工具

  • Charge / Berserk Raygun：远距离补刀与打陷阱。
  • Charge / Berserk Vulcan：低成本单体爆发。
  • Red Sword / Chain Sawd：前期范围清怪。
  • Diska of Braveman + Thirteen：实用远程范围输出。
  • S-Parts ver2.01：机器人 Lv180 前重要的 +15 ATA 来源。

关键 ES / S-Rank

  • Arrest / Hell Slicer 或 J-Cutter：HU 的核心范围特殊武器，优先级最高。
  • Demon's Mechgun：人类 / 纽曼 HU 的高血量单体削血工具。
  • Zalure Gun：HUmar 与机器人在无 FO、Boss 或单刷时使用。
  • Hell Sword / Partisan / Scythe：敌人太近、Slicer 难以利用距离时补位。

终局伤害武器

  • Vjaya：高费用、高上限的多目标爆发。
  • Excalibur：最多三目标，适合冻结后的高血量敌人。
  • Tsumikiri J-Sword：环身、无限垂直高度的范围特殊。
  • Dark Flow：贴身叠五道剑波的最强单体刷怪点爆发。
  • Jizai：处理高 DFP / EVP 单体。

防具与强化

  • Red Ring：大多数场景首选，提供 ATP、ATA 与优秀防御。
  • Anti-Dark Ring：高 EDK 配装核心。
  • D-Parts ver1.01：机器人 +35 ATP。
  • 球顺序参考：Dark Flow → TJS → DoB / Vjaya / Excalibur → Twin Sword → Twin Saber。
  • 属性参考：Machine、Dark 优先；常用组合为 Native / A.Beast 与 Machine / Dark。

RA · Ranger / 枪手

RA 拥有最高的 ATA、优秀射程和最完整的武器特殊工具箱。核心不是站得越远越好，而是找到“足够安全、又能让 Shot / Needle 覆盖最多目标”的距离和角度。

Vanilla 成长路线

时期武器与打法培养重点
Normal–Hard Rifle 在怪物行动前远程点杀，Shot 对付正面扇形怪群，Handgun 用于移动战，Mechgun 留给已硬直或被控制的单体。被逼近时可用 Saber，不必拘泥“只能开枪”。 Mag 以 POW 为主、DEX 为辅；学习用门口、墙角与冻结目标阻挡其他怪物路径。
Very Hard 以高 Hit Shot 清波、Mechgun 爆发、Rifle 控危险目标；开始测试 Arrest、Blizzard、Hell、Demon's 等特殊，而不是只打 Heavy。 武器栏按“范围、单体、控制、Boss”分工，避免背包里全是同一种枪。
Ultimate Shot 仍是主轴，但必须重视 Hit；先冻结 / 麻痹精英再输出。Episode 2 的低 EDK 杂兵适合 Hell，高血目标适合 Demon's。 先建立命中和控场，再追求伤害。机器人用陷阱弥补技能缺失，人类及时补自己的支援技能。

Vanilla 关键习惯：第一枪的价值常常不是伤害，而是让敌人停下、转向或排成 Shot 能覆盖的形状。RA 的安全距离是为队伍争取时间，不是躲在看不到整波怪的位置。

四个角色怎么选

RAmar

全职业最高 ATA，拥有 Lv15 S/D，最容易跨过 Hit 命中阈值。适合想稳定使用各种特殊攻击的玩家。

  • 打法：常驻 S/D，以最高 ATA 先手 Arrest / Hell / Demon's；精英削血后再用 Charge Arms、Mechgun 或 Heaven Striker 收尾。
  • 单刷：射程、命中、Resta 和基础 S/D 组合完整，是最稳的万能单刷角色之一。
  • 误区：最高 ATA 不代表无脑 S-S-S；特殊仍有较低命中修正，要根据距离使用 S-N-S、N-S-S 或 SN glitch。

RAmarl

第二高 ATA，拥有 Lv20 S/D/J/Z，单刷和无 FO 队伍很舒服；高 EVP 也能组成闪避型配置。

  • 打法：先用 J/Z 压低整波怪的攻防，再以 Arms / Needle 清群；高血目标用 Demon's，落单目标由 Heaven Striker 补掉。
  • 组队：无 FO 时承担完整支援；有 FO 时减少重复施法，把优势转为更安全的远程控场。
  • 误区：低 ATP 不应靠过度近战弥补；她的效率来自支援后持续覆盖整波怪。

RAcast

RA 最高 ATP，拥有陷阱；冻结陷阱大幅弥补四名 RA 中最低的 ATA。适合追求范围伤害和爆发。

  • 打法：Freeze Trap 固定高 EVP / 高威胁目标，再用 Baranz / Arms 扫场；单体用高属性 Mechgun 或 Heaven Striker。
  • 组队：有高等级 Shifta 时伤害跃升最大；与 FO 配合，让低级 Gifoie 代为瞬爆陷阱。
  • 误区：Ultimate 下机器人 Demon's 削血受罚，不能照搬人类 RA 的 Demon's 主轴。

RAcaseal

全职业最高 DFP，ATA 略高于 RAcast，拥有陷阱和机器人麻痹加成，控场非常稳定。

  • 打法：以 Spread / Arrest Needle 和陷阱连续控场，队友输出时负责漏怪与侧翼；安全后再接范围伤害。
  • 取舍：比 RAcast 多出的 ATA 有利于特殊命中，但 ATP 与 Freeze Trap 数量较少，更适合控制优先的玩家。
  • 误区：高 DFP 可能让多段攻击不触发击倒而连续吃伤，不能把“面板最硬”理解为可以站桩。

角色专项教程

RAmar · 用最高 ATA 驾驭特殊攻击

核心循环：维持 Lv15 S/D → Rifle / Needle 从远处先控危险怪 → Demon's 削高血目标 → Charge Arms 或 Mechgun 收尾。最高 ATA 的价值是让低命中修正的特殊攻击更可靠，不是让你站在门口持续 Heavy。

  • Arrest：移动快、不能冻或一旦近身就危险的目标先麻痹；Spread Needle 射程短，必须在敌人真正进入覆盖扇面后开枪。
  • Hell：先看 EDK，再决定 Rifle 点杀或 Needle / Shot 扫群；V502 提升效果，但仍要用合适连段和距离。
  • Demon's：只负责快速削血，目标进入普通伤害击杀线就切 Charge / Heaven Striker，避免第三次特殊浪费时间。
  • 单刷：S/D、Resta 与高 ATA 已形成完整闭环；缺 J/Z 时靠 Zalure ES、控制和正确属性弥补。
RAmarl · Lv20 全支援的远程节奏

核心循环：转场 S/D → 新波一次 J/Z 覆盖关键区域 → 用 Needle / Arms 控制并输出 → Demon's 处理高血 → Resta / Anti 后立刻回到射线。RAmarl 的强项是让单刷和无 FO 队伍都保持完整增减益。

  • 施法站位：J/Z 要从能覆盖最多敌人的中距离施放，不要跑进怪群中心追求全覆盖；漏一只通常比被打断更便宜。
  • 输出：ATP 较低时先用 Zalure 和 Demon's 改变击杀线，再用多目标枪收尾；不要拿低伤 Shot 对厚怪连续三段。
  • 有 FO 队伍：让 Lv30 支援覆盖主力，自己只补漏怪、远端队友和紧急治疗，省下动作做 RA 的先手控场。
  • 练习重点：把支援剩余时间和下一波站位一起记，而不是等状态图标消失才在战斗中补。
RAcast · 高 ATP 陷阱枪手完整打法

RAcast 的核心是先用陷阱或枪械特殊固定整波怪，再把最高 RA ATP 转化成四至五目标伤害。他在高等级 Shifta 的多人队伍里提升尤其明显；单刷没有稳定 Shifta，打法会更偏安全控制与特殊攻击。

逐波四步循环

  1. 读形状：五只以内且呈扇面，准备 Arms / Needle；四只排成线或聚在落点，准备 Baranz Launcher；一只高血怪则切 Mechgun / Heaven Striker。
  2. 先固定：能冻结的危险波用 Freeze Trap 或 Frozen Shooter；不能冻结的目标用 Arrest Needle / Raygun。不要在敌人仍横向散开时急着打伤害。
  3. 打主输出：安全时用 N-S-S、S-S-S 的 Charge / Berserk Arms 或 Baranz；命中不稳就先 N 起手，不能一套杀完则提前停手重置距离。
  4. 清漏怪：Heaven Striker / Rifle 点掉扇面外目标；下一波将出现时宁可提前走位，不要用 Shot 慢慢追一只残血。

武器什么时候切

局面首选原因与后续
普通五目标扇面Charge Arms / Berserk Needle覆盖多、动作直接；有 Shifta 和正确属性时优先追求一套清波。
四只高血目标排线Baranz Launcher高伤但弹道和锁定需要站位；先冻结，避免目标移动导致落空。
危险单体Frozen Shooter → Mechgun / HS先冻后打满多段；无须把 Frozen Shooter 当普通输出枪连续射。
不可冻结目标Arrest Needle / Raygun机器人 Paralysis 特殊有加成；麻痹成功后再换伤害武器。
Ep2 低 EDK 杂兵V502 + Hell Needle / Shot / Rifle按密度选择范围；单只目标用 Rifle，别为一个敌人挥霍整套 Shot 动画。
高 HP / 高 DFP物理爆发或队友 Demon's机器人在 Ultimate 使用 Demon's 有减成，通常不要把它当主轴。

陷阱与队伍配合

  • Freeze：给 Baranz / Arms 创造完整射击窗口。若敌人会自然聚团，等半秒再冻往往比生成即冻覆盖更好。
  • Confuse：单刷开荒让高伤怪互打、聚拢；多人 Ultimate 慎用，避免覆盖 Freeze 并打乱敌人路径。
  • Damage：Slime、Rappy 和低 EFR 目标的专项工具，不是普通清怪主力。
  • FO 配合：提前让 FO 用低级 Gifoie 或 Mercurius Rod 特殊铺火环，你放下陷阱即可瞬爆，省去举枪射陷阱的动作。

单刷和组队的不同

单刷

依靠 Mylla & Youlla 取得临时 Shifta；输出不够时不要硬追组队击杀线，改为 Freeze → 安全三段 → 拉开。Hell 有效区域可绕过 ATP 不足。

组队

等 FO 的 Shifta / Zalure 落下再打牺牲特殊，伤害差距很大。你负责整波覆盖和陷阱，HU 贴住厚怪时不要用枪击退、转向它的目标。

最常见错误：因为 HP 和 DFP 高就站桩打满三段。远程职业的防御来自射程与硬直；当怪已进入 Shot 内圈并开始分散，应先 Freeze、换 Needle，或后撤重整射线。

RAcaseal · 控场优先的安卓枪手

核心循环：Arrest / Freeze 固定第一批威胁 → Needle 看守漏怪和侧翼 → 队友开始输出后再用 Arms / Baranz 补范围伤害。她用比 RAcast 更高的 ATA 换取更稳定的特殊攻击，但击杀线更依赖 Shifta、Zalure 与武器属性。

  • 先控再打：一波怪只要有一只冲出覆盖范围，就先用 Raygun / Frozen Shooter 单控，不要为了保持 Shot 连段让它碰到 FO。
  • 陷阱：数量成长和 RAcast 不同，按任务规划 Freeze / Confuse；控住高 EVP 敌人后让 HU 接管高血单体。
  • 高 DFP 陷阱：防御太高时部分多段攻击可能连续命中而不触发击倒，必须仍然尊重固定伤害和属性攻击。
  • 输出补偿：正确敌种属性、Zalure 和 Charge 武器比单纯堆 ATP 单位更能改善清波速度。

目标处理逻辑

  • 普通密集怪：Charge Arms 起步，后期换高 Hit Baranz Launcher；漏怪用 Heaven Striker 点掉。
  • 危险精英：先用 Frozen Shooter / Freeze Trap 控住，再用 Mechgun、Heaven Striker、Excalibur 或 Daylight Scar 爆发。
  • 不能冻结的怪：Dorphon、Girtablulu 等优先用 Spread / Arrest Needle 麻痹，再接 Demon's 或伤害武器。
  • Episode 2 杂兵：大量敌人 EDK 较低，V502 + Hell Rifle / Needle / Shot 是核心路线。
  • 冲锋怪：Twin Blaze 的低级 Gifoie 可阻止 Dark Bringer、Delbiter、Dorphon 首次冲锋。
  • 机器人陷阱：密集强敌用 Freeze Trap；普通怪可用 Confuse Trap 让它们聚团；枪或范围攻击可即时引爆。

FO · Force / 法师

FO 首先是全队效率放大器，其次才是“法术输出”。Lv30 支援技能能显著提高队伍伤害和生存；当前 Ephinea 没有 DMC，合适的技能轰炸、Gifoie 叠层和武器攻击都可以积极使用。

Vanilla 成长路线

时期技能与打法培养重点
Normal–Hard 用低 TP 的 Foie / Zonde / Barta 测敌人弱点，成群敌人再用 Gi / Ra 系；不要用昂贵范围技能轰单只残血怪。手杖主要看 MST 和技能增幅，不是拿来敲怪。 优先 MIND Mag、Mind Material、Mind 单位和较高等级 Resta。随身带足 Fluid，纽曼可在安全转场静止回 TP。
Very Hard 组队优先维持 S/D，并对危险怪补 J/Z；用 Resta、Anti、Reverser 管理队伍。技能抗性高或 TP 紧张时，用 Handgun、Slicer 或带特殊的武器补位。 收集不同元素增幅杖与 Merge，按区域切换；不要把四个单位槽全部锁死为 Mind。
Ultimate 怪物元素抗性拉开差距，先看抗性再选技能。高血高抗目标改用 Demon's，低 EDK 目标用 Hell / Megid；Gifoie 预叠用于已知刷怪点。 在 MST 之外补 HP、抗性和 ATA。纯施法、支援、武器补位三套思路要能随房间切换。

Vanilla 关键习惯：攻击技能看元素抗性而不是 DFP，也不会被 EVP 闪避。第一次打新区域时，小范围试招比固定只按 Rafoie 更可靠；支援技能从 Lv3 起才会覆盖附近队友。

四个角色怎么选

FOmar

FO 最高 ATP，偏战斗法师；Gifoie / Gizonde / Gibarta +30%，Grants +50%，S/D 范围翻倍。ATA 全职业最低，武器 Hit 要求高。

  • 打法：Ep1 / Ep4 以 Gifoie 预叠为轴；高抗性区域切 Excalibur、DoB 或 Demon's / Hell 工具，尽量用命中技巧弥补 ATA。
  • 组队：用 Striker of Chao 扩大 S/D；Resta / Anti 天生范围较小，治疗前要主动靠近队伍。
  • 误区：“战斗法师”不是全程近战；何时从技能切到武器才是 FOmar 的核心判断。

FOmarl

支援取向，S/D/Resta/Anti 范围翻倍，Grants +50%；Slicer 的动作利于命中技巧，Rambling May 射击动作优秀。

  • 打法：先用大范围支援稳住队伍，再按场景用 Rambling May 单体、SoF / DoB 范围削血或 Grants 点杀。
  • 组队：最容易在分散队形中覆盖 S/D、Resta、Anti；应把“少跑位也能及时支援”转化为提前站下一波。
  • 误区：MST 最低且缺少主流范围技能增伤，不适合不看抗性地持续技能轰炸。

FOnewm

Gi 系与 Ra 系伤害均 +30%，是最强范围技能输出；ATA 也足够使用 Demon's Mechgun 等重要工具。

  • 打法：预叠 Gifoie，开波后按弱点选择 Rafoie / Rabarta / Razonde；抗性高的精英换 Demon's Mechgun、Bringer's Rifle 或 Excalibur。
  • 单刷：Ep1 / Ep4 技能覆盖极强，静止时天然回 TP；通过换增幅杖与 Merge 追求击杀阈值。
  • 误区：最强 Nuker 也不等于只按 Ra 系；Ultimate Ep2 尤其需要武器特殊和支援。

FOnewearl

最高 MST、TP 与 FO 最高 ATA，基础三系 +30%，Resta/Anti 范围翻倍，Megid 可贯穿；ATP 和 HP 很低，早期可用 Heavenly/HP 防止固定伤害秒杀。

  • 打法：利用 Foie / Zonde / Barta 高单体效率点杀，直线怪群用 Barta 或贯穿 Megid；高血目标以高 ATA 的 Demon's / Hell 武器削血。
  • 组队：强 Resta / Anti 与长范围让她能兼顾攻击和救场；优先保持安全距离，避免低 HP 被固定伤害秒杀。
  • 误区：最高 MST 不代表范围清图一定强于 FOnewm；她更擅长基础系、Megid、治疗和武器特殊。

角色专项教程

FOmar · 技能与武器混合的近中距离法师

核心循环：Striker of Chao 扩 S/D → Glide Divine 补 J/Z → Gifoie 预叠并整形 → 技能抗性低的怪继续 Gi / Ra → 高抗厚怪切 Demon's、DoB 或 Excalibur。FOmar 的 ATP 是“遇到技能不合适时有第二答案”，不是要求全程近战。

  • ATA 管理：全职业最低 ATA 使 Hit、Smartlink 和控制窗口非常关键；先用 Slicer 的命中技巧或 SoF 削血，再决定是否贴身。
  • 支援范围：S/D 有职业加成，Resta / Anti 没有；治疗必须主动靠近，而不是站远处看着短范围落空。
  • 技能：利用 Gifoie / Gizonde / Gibarta 加成处理合适抗性的怪；不要看到职业加成就忽略敌人元素抗性。
  • 练习重点：每波强制做一次“技能还是武器”的选择，并记录哪一种更快到击杀线。
FOmarl · 大范围支援与稳定武器补位

核心循环：在转场用扩大范围的 S/D 覆盖队伍 → 刷怪后 J/Z → 观察 HU/RA 是否需要 Resta / Anti → 空档用 Rambling May、SoF / DoB 或 Grants 清目标。她的强项是用更少跑位完成支援，把省出的时间转成稳定输出。

  • 站位：保持在队伍中后段，让扩大范围同时碰到前排与后排；不要为了追一只怪离开治疗覆盖中心。
  • 物理补位：Rambling May 处理单体，Slicer 动作适合特殊攻击；冻结或麻痹后再近身，较低 ATA 仍需高 Hit。
  • Grants：职业加成很高，但只在光抗性与目标数量合适时使用；多只普通怪仍应比较范围技能或武器效率。
  • 误区：支援型不等于持续 Resta。队友安全时，提前走位、J/Z 和削血都比过量治疗更有价值。
FOnewm · 范围法术输出完整打法

FOnewm 的 Gi / Ra 系自带 30% 伤害加成，擅长 Episode 1 与 4 的范围清图；但真正的进阶点是在技能、Demon's / Hell 和物理攻击之间快速切换。Ultimate Episode 2 大量目标技术抗性高，继续只按 Rafoie 会同时浪费 TP 和队伍时间。

开波前后的六步循环

  1. 转场：检查 S/D，不在战斗已经开始后才补。换 Striker of Chao 扩范围，施放完立即换回输出武器。
  2. 预判：已知刷怪点先空手快速叠 Gifoie；火环生成后再换 Magical Piece / Gifoie Merge，伤害按命中敌人时的装备计算。
  3. 开波：对高威胁目标补 J/Z；机器人正放陷阱时用低级 Gifoie 帮助瞬爆，但别用状态技能覆盖已冻结目标。
  4. 试抗性：不熟的区域先用一次合适元素观察伤害，再决定 Ra / Gi 连发；不要仅凭敌人颜色猜弱点。
  5. 切解法：技能伤害高就继续施法；高 HP / 高抗目标改 Demon's；低 EDK 敌人改 Hell / Megid；只剩单体残血则用基础系或武器补刀。
  6. 收波：队友安全且目标已有人处理，提前站下一刷怪点；FOnewm 最容易因“多补一发法术”错过预叠窗口。

技能选择不是“哪个范围大”

技能适合不适合
Gifoie固定刷怪点、连续多波、阻止首次冲锋、提供持续硬直追着已散开的怪现放；未预判时施法启动慢
Rafoie火抗低、分布宽、需要即时全范围命中的杂兵单只厚怪或火抗很高的 Ep2 目标

基础三系单体残血、远距离点杀、精确控 TP 成本大量聚团怪；FOnewm 没有 FOnewearl 的基础系职业加成
Megid低 EDK 且排成直线的目标；或无需范围技能伤害的场景高 EDK 目标和需要稳定即时控制的危险波

四套快捷切装

支援套

Striker of Chao 施 S/D，Glide Divine 施 J/Z；支援完成立刻换下，不要拿低输出支援杖打一整波。

Gifoie 套

Magical Piece + Gifoie Merge / Ignition Cloak。预叠时可先空手追求施法动作，再在火环命中前切回增幅。

Ra 系套

Psycho Wand 配对应 Merge；先确认减少 TP 消耗带来的 HP 成本可控，并按区域元素切盾。

武器特殊套

Smartlink + V502，携带 Demon's Mechgun / Bringer's Rifle、SoF、Hell J-Cutter。把高抗性目标视作另一种职业在处理。

单刷与组队

  • 单刷 Ep1 / Ep4：用门口、刷怪点和 Gifoie 预叠形成安全区；纽曼静止回 TP 只用于真正安全的转场。通过换杖和 Merge 追求少一发的击杀阈值，比盲堆 MST 更实际。
  • 组队：第一职责仍是 Lv30 S/D/J/Z 与救场。支援稳定后才以范围技能提高总输出；队友已能一套清波时，多打一发慢法术反而会让你错过下一波站位。
  • Ep2：默认先怀疑技能效率，测试伤害后及时切 Hell、Demon's 或 Excalibur。FOnewm 180 ATA 足以使用重要特殊武器，但仍需要 Hit、Smartlink 和正确连段。
  • Boss：按部位数与抗性选择技能或多段物理；不要把清杂用的 Ra 系配置原样带进所有 Boss。

15 分钟练习法

  1. 选一段熟悉任务，每个房间只记录“最省施法次数的元素”，建立区域抗性直觉。
  2. 练空手 Gifoie → 切 Magical Piece / Merge，让火环在装备切好后命中。
  3. 遇到第一只高抗厚怪，强制切 Demon's 削血再收尾，比较与纯法术的时间和 TP。
  4. 每波结束前停止追残血，由队友补刀，自己提前站下一波并刷新支援。
FOnewearl · 基础系、贯穿 Megid 与精准特殊

核心循环：保持远距离支援 → 用带 30% 职业加成的 Foie / Zonde / Barta 精确点杀 → 直线低 EDK 怪用贯穿 Megid → 高 HP 高抗怪以较高 ATA 的 Demon's / Hell 武器处理。她是单体法术和精确工具专家，不是缩小版 FOnewm。

  • 基础系：单体、远距离、低 TP 成本时优先；Barta 可贯穿直线，站位能把单体技能变成多目标收益。
  • Megid：FOnewearl 自带贯穿，先把怪拉成直线再施放；目标 EDK 高时立即换解法，不要靠连续赌概率拖延。
  • 生存：低 HP 是硬限制。按区域准备 HP / 抗性单位，先保证不会被固定伤害秒杀，再谈多一个 Mind 单位。
  • 组队：扩大 Resta / Anti 适合远距离救场；治疗后马上回到安全射线，不要跟着 HU 进入怪群。

每波怪的优先顺序

  1. 开波前：确认 S/D 剩余时间，移动到能快速覆盖队伍与刷怪点的位置。
  2. 开波瞬间：对高威胁、高 DFP 敌人补 J/Z；机器队友放陷阱时可用低级 Gifoie 协助瞬爆。
  3. 决定输出方式：可预判刷怪点就叠 Gifoie；密集怪用合适的 Ra/Gi 技能；高 HP / 高抗性目标换 Demon's、SoF 或物理武器。
  4. 观察队伍：及时 Resta / Anti / Reverser，但别为了“看起来很忙”打乱冻怪和队友已安排好的分工。
  5. 转场：当前波无需补位时，提前去下一波站位，这是 FO 最容易创造的隐形收益。

技能使用

  • Gifoie：可在刷怪前叠加，伤害在火环碰到敌人时计算。可空手快速施放，敌人出现前再换 Magical Piece / Gifoie Merge 获取增伤。
  • Rafoie / Razonde：用于广域清杂和硬直控场；尤其现行无 DMC 后，合适场景可放心输出。
  • Rabarta：Pre-Ultimate 冻结率 40%，Ultimate 为 20%；缺少冻结陷阱时很有价值。
  • Gizonde：施放速度快，可开箱、清小怪，并用于锁住 Vol Opt 第一阶段。
  • Megid / Hell：Episode 2 低 EDK 杂兵和 Episode 4 特定敌人很有效；FOnewearl 的 Megid 自带贯穿。
  • 低级 Gifoie：Mercurius Rod 特殊可停止 Delbiter、Dark Bringer、Dorphon、Zu 的首次冲锋。

装备路线

支援与技能增幅

  • Striker of Chao：S/D 范围翻倍，常驻携带。
  • Glide Divine：J/Z 范围翻倍。
  • Magical Piece：Gi 系 +30%，Gifoie 核心。
  • Psycho Wand：Ra 系 +30%，单刷和技能输出配置更重要。
  • Mercurius Rod：Grants +30%，低级 Gifoie 特殊可拦冲锋。

武器特殊与物理补位

  • Slicer of Fanatic：三目标无衰减 Demon's。
  • Bringer's Rifle：长距离单体 Demon's。
  • Hell Raygun / ES J-Cutter：Episode 2 单体 / 范围即死。
  • Demon's ES Mechgun：FOnewm / FOnewearl 处理高血高防目标。
  • DoB + Thirteen / Excalibur：需要物理伤害时使用。

核心单位

  • V801：法术施放速度核心。
  • V101：武器攻击速度与综合属性。
  • Smartlink：消除非 RA 使用远程武器的距离命中惩罚。
  • V502：Episode 2 的 Hell / 状态特殊核心。
  • Technique 单位：技能盘未到 Lv30 时按需补级。

铠甲与盾

  • Ignition Cloak + Gifoie Merge：Gifoie 伤害核心组合。
  • Thirteen：使用 DoB 时切换。
  • Dress Plate / Anti-Dark Ring：高 EDK 配置。
  • Red Ring：武器型 FO 的 ATA 与综合属性终局选择。
  • 各类 Merge：根据关卡主要技能切换，不要一件盾穿到底。
