什么是苦痛模式?

苦痛模式是Ephinea独有的一种类似于暗黑3的自我挑战模式，允许玩家在更高的怪物难度下挑战自己。80级以上的角色可以在U难度下通过大厅柜台选择3个苦痛等级中的任意一个。

当你提升苦痛难度的等级，那么怪物和奖励也会相应的提升和加强:

奖励提升:

  • 掉落武器命中提升
  • 经验提升
  • 稀有掉落率提升
  • 精神力(MST)提升

怪物强化:

  • 伤害提升
  • 攻击力(ATP)提升
  • 命中(ATA)提升
  • 防御(DFP)不变
  • 回避(EVP)提升
  • 暴击(LCK)提升
  • 暗抗(EDK)提升
  • EX抗性(ESP)提升

苦痛游戏等级可以在创建U难度游戏时在大厅柜台设置。对于每个苦痛难度等级，你可以在大厅柜台按TAB键看到怪物、掉落和经验等提升的描述。

颜色ID特产如下:

  • 镉绿 -> 散弹
  • 翠绿 -> 步枪
  • 天清 -> 大剑
  • 纯蓝 -> 长刀
  • 淡紫 -> 机枪
  • 粉红 -> 魔杖
  • 真红 -> 单手剑/手杖 (各50)
  • 橙黄 -> 匕首
  • 金黄 -> 手枪/长杖 (各50)
  • 羽白 -> 投刃

常规武器掉率3是对应低级区域，4是对应高级区域，5对应红盒。(0、1、2、3、4在普通模式都覆盖、5只是针对红盒)

低级区域: 森林, 洞窟, 神殿, 宇宙船, 陨石坑

高级区域: 坑道, 遗迹, 中央管理区, 海底, 塔, 沙漠
